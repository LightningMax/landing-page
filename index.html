Comment utiliser Linux sur Windows?

Qu'est-ce que c'est WSL ?

Le Sous-Système Linux pour Windows permet d’exécuter une distribution Linux directement sur Windows, ce qui vous permettra d’utiliser des commandes, applications et outils linux sans besoin de faire un dualboot ou installer une machine virtuelle traditionnelle.
